How to register the dll to avoid manual loading for each session of AutoCAD ?

If you want to avoid loading the app manually for each session of AutoCAD, try the following.

  • Download this lisp file and save to your computer.

  • At the command line type APPLOAD and select the lisp file you downloaded or drag/drop the lisp file to AutoCAD

  • Click Load and close the dialog.

  • Then at the command line type LoadDwgPurge and enter
